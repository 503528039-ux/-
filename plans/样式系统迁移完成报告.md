## 📋 样式使用指南

### 在 Vue 组件中使用

#### 方法 1: 直接使用 CSS 类
```vue
<template>
  <div class="a4-page">
    <header class="page-header">
      <div class="header-title">2026 工程产品手册</div>
    </header>
    
    <div class="page-content">
      <h2 class="section-title">产品系列</h2>
      <div class="section-subtitle">Product Line</div>
      
      <div class="grid-6">
        <div class="product-card">
          <div class="card-img-box">
            <img :src="product.image" />
          </div>
          <div class="card-info">
            <div class="card-title">{{ product.name }}</div>
            <div class="card-model">{{ product.model }}</div>
          </div>
        </div>
      </div>
    </div>
    
    <div class="page-footer">- PAGE 09 -</div>
  </div>
</template>
```

#### 方法 2: CSS 变量在 scoped 样式中使用
```vue
<style scoped>
.custom-title {
  color: var(--archie-purple);
  font-size: 28px;
  margin-bottom: var(--spacing-md);
}

.custom-box {
  background: var(--luxury-cream);
  padding: var(--page-padding);
  border-radius: var(--border-radius-md);
}
</style>
```

## 🔄 与 Tailwind CSS 兼容性

所有提取的样式与现有 Tailwind 配置完全兼容:

1. **CSS 变量优先**: 自定义样式使用 CSS 变量, Tailwind 可随时覆盖
2. **BEM 命名**: 组件类使用明确的 BEM 命名，避免与 Tailwind 冲突
3. **层叠顺序**: 
   ```css
   @tailwind base;     ← 基础重置
   @tailwind components; ← Tailwind 组件
   (核心组件样式)       ← 自定义组件
   @tailwind utilities; ← Tailwind 工具类 (优先级最高)
   ```

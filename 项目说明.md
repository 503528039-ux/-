# 雅洁五金2026年工程图册项目说明

## 📋 项目概述

这是一个**广东雅洁五金有限公司**的2026年工程产品图册在线编辑系统。项目基于Vue 3 + Vite构建，提供了可视化的产品图册编辑、排版和导出功能。

**通俗比喻**：这就像一个"数字化的产品画册工厂"，你可以像拼图一样拖拽产品图片和文字，快速制作出专业的工程产品图册。

## 🛠️ 技术栈

- **前端框架**: Vue 3 + Composition API
- **构建工具**: Vite 5.3.0
- **状态管理**: Pinia 2.1.7
- **UI样式**: Tailwind CSS 3.4.17
- **拖拽功能**: vuedraggable 4.1.0
- **数据处理**: xlsx 0.18.5 (Excel处理)
- **导出功能**: jspdf 2.5.1 + html2canvas 1.4.1

## 🎯 核心功能

### 1. **可视化编辑**
   - 拖拽式页面布局（像PPT一样简单）
   - 实时预览效果
   - 支持A4/A3纸张尺寸切换

### 2. **产品管理**
   - 批量导入产品图片（支持拖拽）
   - 自动匹配产品型号和图片
   - 产品信息编辑（型号、材质、规格等）

### 3. **智能排版**
   - 瑞士极简风格封面设计
   - 自动页码编号
   - 网格对齐辅助线

### 4. **导出功能**
   - 支持PDF导出
   - 打印预览模式
   - 高质量图片输出

## 📊 产品数据规模

- **产品图片**: 超过75个产品系列，分布在4个完整包中
- **产品类型**: 锁具、把手、合页、配件等五金产品
- **图片格式**: JPG + PNG（含透明背景）
- **数据源**: Excel产品清单（总产品清单.xls）

## 💡 使用指南

### 基础操作
1. **添加页面**: 从侧边栏选择页面类型（封面、产品页、品牌介绍等）
2. **拖拽元素**: 将产品图片拖到页面中任意位置
3. **编辑内容**: 点击文字区域直接编辑
4. **调整大小**: 拖动元素角落的调整手柄

### 高级功能
- **批量导入**: 直接将多个产品图片拖到页面区域
- **撤销/重做**: Ctrl+Z / Ctrl+Y
- **复制粘贴**: Ctrl+C / Ctrl+V
- **页面尺寸**: 右键页面选择A4竖版/横版/A3横版

### 导出图册
1. 点击"预览"按钮进入打印模式
2. 检查排版和内容
3. 使用浏览器打印功能导出PDF

## 🎓 小贴士（最佳实践）

**比喻理解**: 把这个系统想象成一个"数字化的乐高积木"：
- **页面** = 底板
- **产品块** = 积木块  
- **拖拽** = 拼装
- **导出** = 拍照保存作品

**实用技巧**:
1. 先整理好产品图片和Excel清单，再批量导入
2. 使用网格对齐功能保持版面整洁
3. 定期保存工作进度（系统有自动保存）
4. 导出前务必进入预览模式检查排版
